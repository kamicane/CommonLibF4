# CommonLibF4

[![C++23](https://img.shields.io/static/v1?label=Standard&message=C%2B%2B23&color=blue&logo=c%2B%2B&&logoColor=white&style=flat)](https://en.cppreference.com/w/cpp/compiler_support)
[![Platform](https://img.shields.io/static/v1?label=Platform&message=Windows%20x64&color=dimgray&style=flat)](#)

This is a fork of [LucaDotGit's CommonLibF4](https://github.com/LucaDotGit/CommonLibF4), with AE ids added in from [libxse/commonlibf4](https://github.com/libxse/commonlibf4).

Goal is to have a CommonLibF4 that can produce dlls that are compatible with both OldGen (1.10.163) and Anniversary Edition.
